RAMP-GUI Standalone python file
Disclaimer : This project is open-source. Interested users are therefore invited to test, comment or contribute to the tool.

Requirement : Python 3 and above
Package required : rampdemand, PyQt5 and all other pre installed standard packages

To modify the GUI according to your needs, you may use RAMP.py file.
In the begining, inside .py file there are list of imported packages. All those required packages must be installed in python.

This Project was developed using Qt Designer & PyQt5.
